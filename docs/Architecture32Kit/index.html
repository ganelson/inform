  • Architecture32Kit

Inter-level support for compiling Inform programs to 32-bit architectures. This is version 1.

  • Capabilities Template - Miscellaneous capabilities of the 16-bit architecture.

  • Glk Template - Glk is a fully-featured input-output model, designed by Andrew Plotkin for use with the Glulx virtual machine, but not used only for that. All 32-bit Inform programs use Glk.

  • Startup Template - How programs for this architecture start up the virtual machine.

  • State Template - Saving, restoring, restarting and verifying the program from within itself.

  • Mathematics Template - Mathematical functions, especially for real numbers.

  • Character Set Template - Code relating to the Unicode character set.

  • Input Output Template - Access to the keyboard and to textual windows.

  • File Input Output Template - Access to external files.

  • Veneer Template - Veneer code for use by the Inform 6 compiler with the Glulx target.
